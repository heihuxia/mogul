- :dog: mock 服务器: json-server, faker 快速模拟数据

## install

```bash
yarn add @mogul/components
```

> 强烈推荐和 [create-mogul-app](https://freshesx.github.io/mogul-website/docs/create-mogul-app) 脚手架 一起使用, 

```bash
npx @mogul/create-mogul-app <project-name>
```
